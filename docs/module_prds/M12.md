# M12 — Sharing Module

**Priority:** P0 | **Phase:** 1 | **Complexity:** Low

## Description
Generate shareable links for trips. Public links, invite-only links, and trip duplication from shared views.

## Acceptance Criteria
- [ ] Generate public share link (randomized token, not sequential ID)
- [ ] Generate invite-only share link
- [ ] Share link with optional expiration
- [ ] Public shared trip view page (read-only, no auth required)
- [ ] Duplicate shared trip into own account
- [ ] Revoke share link
- [ ] Copy-to-clipboard share URL
- [ ] Social sharing meta tags (Open Graph) for shared trip pages

## Database Touchpoints
- `shared_links` table (CRUD)
- `trips` (read for shared view, write for duplicate)

## API Touchpoints
- POST /api/v1/trips/:id/share
- GET /api/v1/share/:token
- DELETE /api/v1/share/:linkId
- POST /api/v1/share/:token/duplicate

## Dependencies
- M1, M2, M5
