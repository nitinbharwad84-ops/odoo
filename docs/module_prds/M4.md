# M4 — Dashboard Module

**Priority:** P0 | **Phase:** 1 | **Complexity:** Low

## Description
Primary user home workspace showing recent trips, upcoming trips, create trip CTA, quick AI planner entry, budget alerts, and trip summaries.

## Acceptance Criteria
- [ ] Welcome panel with user greeting
- [ ] Create trip CTA button (prominent)
- [ ] Recent trips grid/list (last 6 trips)
- [ ] Upcoming trips section (trips with future start_date)
- [ ] Quick AI planner CTA
- [ ] Budget alerts for active trips exceeding target
- [ ] Empty state for new users (onboarding prompt)
- [ ] Responsive layout (mobile/tablet/desktop)

## Database Touchpoints
- `trips` table (read — user's trips)
- `budgets` table (read — alert calculation)

## API Touchpoints
- GET /api/v1/trips (with filters: recent, upcoming)

## Dependencies
- M1 (Foundation), M2 (Authentication), M3 (Profile)
