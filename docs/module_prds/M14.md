# M14 — Notifications Module

**Priority:** P1 | **Phase:** 1 | **Complexity:** Low

## Description
In-app notification system for collaboration invites, budget alerts, trip reminders, and AI completion notifications.

## Acceptance Criteria
- [ ] Notification bell icon with unread count in header
- [ ] Notification center page listing all notifications
- [ ] Mark individual as read
- [ ] Mark all as read
- [ ] Notification types: collab_invite, budget_alert, trip_reminder, ai_complete
- [ ] Click notification to navigate to relevant page
- [ ] Email notifications for collaboration invites (via Resend)

## Database Touchpoints
- `notifications` table (CRUD)

## API Touchpoints
- GET /api/v1/notifications
- PATCH /api/v1/notifications/:id/read
- POST /api/v1/notifications/read-all

## Dependencies
- M1, M2
