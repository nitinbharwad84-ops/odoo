# M15 — Community Module (Phase 2)

**Priority:** P2 | **Phase:** 2 | **Complexity:** Medium

## Description
Public itinerary feed, discover shared trips, like/comment/save, copy trips, follow users, creator discovery.

## Acceptance Criteria
- [ ] Public community feed page
- [ ] Post itinerary to community
- [ ] Like/unlike posts
- [ ] Comment on posts
- [ ] Save trip templates
- [ ] Copy/duplicate community trips
- [ ] Follow/unfollow users
- [ ] User profile public view
- [ ] Feed sorting (recent, popular, trending)

## Database Touchpoints
- `community_posts`, `comments`, `likes`, `followers` (CRUD)

## API Touchpoints
- GET /api/v1/community/feed
- POST /api/v1/community/posts
- POST /api/v1/community/comments
- POST /api/v1/community/likes

## Dependencies
- M1, M2, M5, M12
