# M8 — Budget Planner Module

**Priority:** P0 | **Phase:** 1 | **Complexity:** Medium

## Description
Trip budget management with target setting, category breakdown, estimated vs actual spending, daily summaries, alerts, and charts.

## Acceptance Criteria
- [ ] Set trip budget target
- [ ] Category breakdown (flights, accommodation, food, activities, transport, shopping, insurance, emergency, misc)
- [ ] Estimated vs actual amount per category
- [ ] Daily spend summary calculation
- [ ] Budget alerts (over budget, high daily spend, category overspend)
- [ ] Pie chart of budget allocation (Recharts)
- [ ] Bar chart comparing estimated vs actual
- [ ] Total budget summary with remaining amount
- [ ] Edit individual category amounts inline
- [ ] Currency display based on trip currency

## Database Touchpoints
- `budgets` table (CRUD)
- `trips` table (read budget_target, currency)

## API Touchpoints
- GET /api/v1/trips/:id/budget
- POST /api/v1/trips/:id/budget
- PATCH /api/v1/budget/:id

## Dependencies
- M1, M2, M5 (Trip Management)
