# M13 — AI Services Module

**Priority:** P1 | **Phase:** 1 | **Complexity:** High

## Description
AI-powered trip planning, budget prediction, and packing list generation using Google Gemini API (primary) and Groq (fallback). Structured JSON outputs, rate limiting, usage logging.

## Acceptance Criteria
- [ ] AI Trip Planner: generate full itinerary from natural language prompt
- [ ] AI Budget Predictor: estimate budget categories from trip context
- [ ] AI Packing Generator: generate categorized packing list from trip context
- [ ] Structured JSON output with Zod schema validation
- [ ] Loading state with streaming indicator
- [ ] Accept/regenerate/edit flow for AI outputs
- [ ] Convert AI-generated trip to editable trip
- [ ] Fallback from Gemini → Groq on provider failure
- [ ] Rate limiting: 10 AI calls/day per user (free tier)
- [ ] Usage logging to ai_usage_logs table
- [ ] Error handling for: timeout, rate limit, invalid response, provider outage
- [ ] Prompt templates with system/context/user layers

## Database Touchpoints
- `ai_usage_logs` table (write)
- `trips`, `trip_stops`, `trip_activities` (write — on accept)
- `budgets` (write — on accept budget)
- `packing_items` (write — on accept packing)

## API Touchpoints
- POST /api/v1/ai/plan-trip
- POST /api/v1/ai/budget
- POST /api/v1/ai/packing

## Dependencies
- M1, M2, M5, M6, M8, M9
