# M10 — Notes & Journal Module

**Priority:** P1 | **Phase:** 1 | **Complexity:** Low

## Description
Trip notes, daily journal entries, reminders, hotel details, travel/emergency contacts. Rich text editing via Tiptap.

## Acceptance Criteria
- [ ] Create notes with rich text (Tiptap editor)
- [ ] Note types: general, daily, reminder, hotel, contact, emergency, journal
- [ ] Link notes to specific days (linked_day field)
- [ ] Edit and delete notes
- [ ] Notes list view with type filters
- [ ] Reminder notes with visual indicator

## Database Touchpoints
- `notes` table (CRUD)

## API Touchpoints
- GET/POST /api/v1/trips/:id/notes
- PATCH/DELETE /api/v1/notes/:id

## Dependencies
- M1, M2, M5
