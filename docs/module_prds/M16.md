# M16 — Admin Panel (Phase 2)

**Priority:** P2 | **Phase:** 2 | **Complexity:** Medium

## Description
Admin dashboard for user management, moderation, analytics overview, AI usage monitoring, and abuse detection.

## Acceptance Criteria
- [ ] Admin dashboard with key metrics
- [ ] User management (list, search, suspend, delete)
- [ ] Community moderation (review flagged posts, remove content)
- [ ] AI usage dashboard (total tokens, cost, failures)
- [ ] Basic analytics view (signups, trips created, AI usage)
- [ ] Audit log viewer

## Database Touchpoints
- `users`, `profiles` (read/write)
- `community_posts`, `comments` (moderation)
- `ai_usage_logs` (read)
- `audit_logs` (read)

## API Touchpoints
- Admin-only endpoints (to be defined in Phase 2)

## Dependencies
- M1, M2, M15 (Community)
