# M1 — Project Foundation & PWA Setup

**Priority:** P0 (Must Ship) | **Phase:** 1 | **Complexity:** Medium

## Description
Initialize the Next.js 14 project with TypeScript, Tailwind CSS, shadcn/ui, and PWA support. Set up the design system, layout components, dual-database connection (Supabase + Local PostgreSQL via Prisma), and core infrastructure (providers, middleware, error boundaries).

## Acceptance Criteria
- [ ] Next.js 14 App Router project with TypeScript strict mode
- [ ] Tailwind CSS + shadcn/ui configured with custom theme (dark/light mode)
- [ ] PWA manifest, service worker, offline page functional
- [ ] Prisma schema with dual-database connection (Supabase + Local PG)
- [ ] DatabaseService with health monitoring and failover logic
- [ ] Supabase client (browser + server) configured
- [ ] App shell layout (sidebar, header, mobile nav)
- [ ] Auth middleware protecting authenticated routes
- [ ] Error boundary and 404/500 pages
- [ ] SEO metadata configured for public pages
- [ ] PostHog + Sentry initialized
