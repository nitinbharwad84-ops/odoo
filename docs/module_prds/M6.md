# M6 — Itinerary Builder & Viewing Module

**Priority:** P0 | **Phase:** 1 | **Complexity:** High

## Description
Core product workflow. Build multi-city itineraries with trip stops, day-wise activity planning, drag-and-drop reordering, inline editing, and multiple view modes (timeline, day view, list, calendar, map).

## Acceptance Criteria
- [ ] Add trip stop (city, country, dates, timezone, transport cost/time)
- [ ] Remove trip stop with confirmation
- [ ] Reorder stops via drag-and-drop (@dnd-kit)
- [ ] Assign arrival/departure dates to stops
- [ ] Add activities to stops (title, description, category, cost, duration, time slot)
- [ ] Remove/edit activities inline
- [ ] Day-wise planning view (activities grouped by day)
- [ ] Trip notes per stop
- [ ] Timeline view (vertical timeline of stops + activities)
- [ ] Day view (single day detail)
- [ ] List view (compact table)
- [ ] Calendar view (month calendar with stops highlighted)
- [ ] Map view (Leaflet map with stop markers and route lines)
- [ ] Expand/collapse sections
- [ ] Trip summary panel (total days, total stops, total estimated cost)
- [ ] Print-friendly layout

## Database Touchpoints
- `trip_stops` table (CRUD + reorder)
- `trip_activities` table (CRUD + reorder)
- `activities` table (reference data)

## API Touchpoints
- GET/POST /api/v1/trips/:id/stops
- PATCH/DELETE /api/v1/stops/:id
- POST /api/v1/stops/:id/activities
- PATCH/DELETE /api/v1/trip-activities/:id
- PATCH /api/v1/stops/reorder

## Dependencies
- M1 (Foundation), M2 (Auth), M5 (Trip Management)
