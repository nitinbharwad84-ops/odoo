# M9 — Packing Checklist Module

**Priority:** P1 | **Phase:** 1 | **Complexity:** Low

## Description
Manual packing list creation and AI-powered packing generation. Categorized checklist with pack/unpack toggling.

## Acceptance Criteria
- [ ] Manual item creation (name, category)
- [ ] AI packing generation based on trip context (destination, season, duration, trip type)
- [ ] Categories: clothing, electronics, documents, hygiene, medicine, accessories, travel_gear
- [ ] Toggle packed/unpacked per item
- [ ] Duplicate checklist from another trip
- [ ] Reset checklist (unpack all)
- [ ] Delete individual items
- [ ] Progress indicator (X of Y packed)

## Database Touchpoints
- `packing_items` table (CRUD)

## API Touchpoints
- GET/POST /api/v1/trips/:id/packing
- PATCH/DELETE /api/v1/packing/:id

## Dependencies
- M1, M2, M5, M13 (AI Services for generation)
